Программа для создания массива строк из исходного массива строк, длина которых не более 3 символов.
Программа считывает пользовательский ввод массива.
1. Метод CreateArrayString создает массив из данных, введённых пользователем
2. Метод PrintArray выводит массив в консоли.
3. Метод NewArray находит длинну нового массива, а далее создаёт новый массив.
